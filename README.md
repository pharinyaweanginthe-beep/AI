# AI Technologies - เครื่องมือปัญญาประดิษฐ์อัจฉริยะสำหรับอนาคต

![AI Technologies Banner](images/hero-ai.png)

## 🌐 Live Demo
ไปที่เว็บไซต์: [ai-tawny-rho.vercel.app](https://ai-tawny-rho.vercel.app/)

เว็บไซต์ **AI Technologies** (AITech) เป็นแพลตฟอร์มที่นำเสนอข้อมูลเกี่ยวกับเทคโนโลยีปัญญาประดิษฐ์ (Artificial Intelligence) และโซลูชันด้าน AI สำหรับธุรกิจยุคใหม่ ออกแบบมาให้มีความทันสมัย รองรับการใช้งานทุกอุปกรณ์ (Responsive Design) และมีเอฟเฟกต์การแสดงผลที่สวยงาม

## 🌟 ฟีเจอร์หลัก (Key Features)

- **หน้าแรก (Home):** นำเสนอภาพรวมของเทคโนโลยี AI และโปรโมชั่นโซลูชันสำหรับธุรกิจ
- **เกี่ยวกับ (About):** ข้อมูลเชิงลึกเกี่ยวกับวิสัยทัศน์และพันธกิจของ AI Technologies
- **AI Applications:** รวบรวมตัวอย่างการนำ AI ไปใช้งานในด้านต่างๆ เช่น การแพทย์, อุตสาหกรรม และการศึกษา
- **ข่าวสารและบทความ (News & Articles):** อัปเดตเทรนด์ AI ล่าสุดและบทความที่เป็นประโยชน์
- **Responsive Design:** แสดงผลได้อย่างสวยงามบนมือถือ แท็บเล็ต และคอมพิวเตอร์
- **Interactivity:** มีระบบ Navigation bar ที่เปลี่ยนตามการเลื่อน (Scroll) และแอนิเมชันตอนเลื่อนดูเนื้อหา

## 🛠️ เทคโนโลยีที่ใช้ (Technologies Used)

- **HTML5:** โครงสร้างเว็บที่เป็นมิตรกับ SEO (Semantic HTML)
- **Vanilla CSS3:** ดีไซน์สไตล์ Modern Dark Theme พร้อมเอฟเฟกต์ Gradient และ Glassmorphism
- **JavaScript (ES6+):** ควบคุมส่วนโต้ตอบ (Interactivity) และแอนิเมชันต่างๆ
- **Google Fonts:** ใช้ฟอนต์ 'Inter' และ 'Outfit' เพื่อความสวยงามระดับพรีเมียม

## 📂 โครงสร้างโปรเจกต์ (Project Structure)

```text
/
├── index.html          # หน้าแรก
├── about.html          # หน้าเกี่ยวกับเรา
├── applications.html   # หน้าแอปพลิเคชัน AI
├── news.html           # หน้าข่าวสาร (หน้าหลัก)
├── news2.html          # หน้าบทความย่อย
├── css/
│   └── style.css       # ไฟล์สไตล์หลัก
├── js/
│   └── main.js        # ไฟล์ JavaScript
└── images/             # โฟลเดอร์เก็บรูปภาพประกอบ
```

## 🚀 วิธีการใช้งาน (Getting Started)

1.  ดาวน์โหลดหรือ `git clone` โปรเจกต์นี้ลงในเครื่องของคุณ
2.  เปิดไฟล์ `index.html` ผ่านเบราว์เซอร์ (Chrome, Firefox, Safari หรือ Edge)
3.  สนุกกับการสำรวจโลกของ AI!
